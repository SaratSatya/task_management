Project stack:
- Next.js App Router
- MongoDB with Mongoose
- JWT auth in HTTP-only cookies
- bcryptjs for password hashing
- Vercel deployment

Rules:
- Build only the assignment requirements first
- Keep architecture simple and production-ready
- Do not add unnecessary libraries
- Use environment variables for secrets
- Backend must enforce per-user task authorization
- Use structured JSON responses
- Keep validation consistent
- Update README and docs when behavior changes
